Thoughts on Progressive Enhancement

Progressive enhancement in web development is an important concept to consider that is focused on making the content of the site accesible to as many people as possible. This involves starting with a solid foundation in the HTML layer that can run on any browser or device and that can be built upon using CSS and JS. This ensures that anyone can access the content as well as providing a base level that will show if there are issues with the functionality of the CSS and JS. When I first started this class, I was pleasantly surprised by how much attention was given to making sure our code was accessible. It made a lot of sense but I had not considered doing so before this class.

I think progressive enhancement is important in web development because I place a lot of value in accessibility. The content of websites is what matters most and the whole point of making websites is so that people will visit it. If you are shutting out groups of people by not making sure that it is accessible, then people will rightly stop visiting the site. It also helps improve the consistency of quality control of sites. A strong foundational layer of HTML can be understood by other people and helps save time when debugging isues.
